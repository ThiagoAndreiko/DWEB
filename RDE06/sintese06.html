Software de Versionamento

Esse sistema tem por objetivo realizar o controle de versões de um código ou documento.
Sendo possivel gerenciar as versões do código ou documento realizadas por um desenvolvedor ou uma equipe de desenvolvedores.

Tipos de Software de Controle de Versão

Centralizado

Possui repositório central em um servidor com contole de versões, dependente do servidor.

Software Subversion

Distribuído

Possui repositório local com controle de versões.Atualizações no repositório cental independente do sérvidor.

Software Mercurial, Git
