Hiperlinks

É uma referência de um documento em hipertexto para outras partes do documento.

Marcação de hiperlinks

A marcação ocorre utilizando a TAG

Hiperlinks externos ou remotos

Para fazer o uso dela utilizamos a TAG

Hiperlinks internos

São links que apontam locais diferentes dentro da mesma página e posibilita acesso a outos parágrafos ou sesões

Hiperlinks Locais

São links que direcionam para outras páginas

Hiperlinks em Imagens

Para realiza basta a tag imagem com a tag hiperlink

Hiperlinks para Downoad de Arquivos

São links que apontam para arquivos que possibilitam download

Hiperlinks para endereço eletrônico

Este hiperlink aponta um endereço eletrônico.
